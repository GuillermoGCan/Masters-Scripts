R: El comando facets = TRUE nos sirve para separar las grficas de cada serie de tiempo, de forma que se pueda apreciar el comportamiento de cada una en su escala ptima correspondiente. De esta forma se puede apreciar con mayor claridad la correlacin entre las variables.
